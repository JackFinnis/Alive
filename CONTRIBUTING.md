# Contributing

Contributions are welcome! Here's how to get started:

1. Check [TODO.md](TODO.md) for ideas and improvements
2. Fork the repo and create a branch
3. Make your changes
4. Open a pull request

## Setup

This is a visionOS app built with Xcode 16.3+ and Swift 6.0. There is no command-line build pipeline — open `Alive.xcodeproj` and build from Xcode.

> **Note:** 3D models and image assets are not included in this repository. Email [jack@jackfinnis.com](mailto:jack@jackfinnis.com) for access to the asset files needed to build and run the app.
